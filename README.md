### 🎯 O Projeto
Este repositório é um laboratório de **DevOps Culture**. O objetivo principal foi criar uma API resiliente que se auto-gerencia desde o primeiro `git push` até o ambiente de produção na AWS.

### 💎 Diferenciais Estratégicos
* **Zero Downtime:** Configuração preparada para deploys sem interrupção.
* **Segurança First:** Uso de Secrets e permissões restritas via IAM.
* **Imutabilidade:** Cada build gera uma imagem única tagueada via SHA do commit.
